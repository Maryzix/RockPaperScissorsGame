# Rock Paper Scissors - Java

A simple Rock Paper Scissors game developed in Java to practice programming fundamentals and problem-solving skills.

## Features

- User input with `Scanner`
- Random computer choices using `Random`
- Win, loss, and draw detection
- Score tracking
- Continuous gameplay with a menu loop

## Concepts Practiced

- Variables
- Conditional statements (`if` / `else`)
- Loops (`while`)
- User input (`Scanner`)
- Random number generation (`Random`)
- Basic game logic

## Technologies

- Java 8+
- IntelliJ IDEA / VS Code

## How to Run

Compile:

```bash
javac ExercicioJokenpo.java
```

Run:

```bash
java ExercicioJokenpo
```

## Example

```text
=== ROCK PAPER SCISSORS ===

1 - Rock
2 - Paper
3 - Scissors

Choose an option: 1

You chose: Rock
Computer chose: Scissors

You win!
```
